# GeminiCook

> 'Gemini API 개발자 대회' 제출용 (지금은 사용 불가합니다.)
Gemini API를 이용해서 **요리/레시피 관련 기능**을 만드는 개인 프로젝트 템플릿입니다.  
(좋아하는 음식, 가지고 있는 재료 등 이것저것 파악하여 Gemini를 이용하여 레시피를 보여주는 프로그램)

<img width="40%" alt="image" src="https://github.com/user-attachments/assets/845b986f-d7d3-4b01-a180-b3fef1d425b4" />

### 프로그램 설명 영상 : https://youtu.be/FLRkyRzsFpE

## ✨ Features

- 재료 입력 → 추천 레시피 생성
- 인원/알레르기/선호도 반영(매운맛, 채식, 저탄수 등)
- 조리 단계(steps) 구조화 출력
- 장보기 리스트/시간표 생성
- 결과를 Markdown/JSON으로 저장
  
## Code Clone
```
git clone https://github.com/seunghwan94/GeminiCook.git
```

## Download Library
```
pip install -r requirements.txt
```

## start Streamlit
```
streamlit run GeminiCook.py
```
